# Projet DevSecOps – Akaunting

## Description
Ce projet met en place une chaîne DevSecOps autour de l’application Akaunting.
L’objectif est de démontrer l’automatisation CI/CD, la conteneurisation et l’intégration de la sécurité.

## Technologies utilisées
- GitHub Actions (CI/CD)
- Docker (Conteneurisation)
- PHP / Laravel
- Trivy (Analyse de sécurité)

## Pipeline CI/CD
Le pipeline automatise :
- La récupération du code
- L’installation des dépendances
- La vérification de la syntaxe PHP
- La construction de l’image Docker
- L’analyse de sécurité (non bloquante)

## Objectif pédagogique
Ce projet est réalisé dans un cadre académique afin d’appliquer les bonnes pratiques DevSecOps.
Le déploiement cloud sera ajouté ultérieurement.
